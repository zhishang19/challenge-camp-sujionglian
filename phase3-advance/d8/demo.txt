## 样例 1：知识库查询（同义词扩展）

输入：`怎么离线安装.deb软件包？`

流程：
  Node1 [classify] → 意图=knowledge_query, 无PII
  Node2 [retrieve] → bigram搜索 + synonym("deb"→"dpkg") → 命中 K001(score=0.071)
  Node3 [output]  → 格式化返回知识条目正文

输出：
```
[K001] 离线安装 .deb 软件包（办公场景） 麒麟 安装 deb
       步骤: cd到deb所在目录 → sudo dpkg -i xxx.deb
       → sudo apt install -f (如有依赖问题)
```

## 样例 3：PII 过滤 + 知识查询

输入：`liubei@shu.com 帮我把文档导出成PDF`

流程：
  Node1 [classify] → PII检测到 email: liubei@shu.com → 脱敏为[EMAIL]
                    意图=knowledge_query
  Node2 [retrieve] → 搜索 "帮我把文档导出成PDF"... (当前未命中)
  Node3 [output]  → 标注PII已过滤

```

未找到相关知识条目。
```
