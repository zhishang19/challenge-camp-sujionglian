# D10 麒麟操作系统 认知题

## 1. 银河麒麟与 Windows 的主要区别（包管理/软件安装）

| 对比维度 | 银河麒麟 (Linux) | Windows |
|----------|-----------------|---------|
| 包管理器 | apt / dpkg（Debian系） | 无统一包管理器，exe/msi安装 |
| 软件安装 | `sudo apt install xxx` 命令行 | 双击安装包向导 |
| 软件源 | 官方仓库 + PPA | 各自官网下载 |
| 权限模型 | sudo 提权 | UAC 弹窗确认 |
| 文件系统 | ext4/xfs | NTFS |

## 2. 为何强调「国产 OS + Agent 记忆模块」

麒麟赛题强调国产 OS + Agent 记忆模块，是因为银河麒麟作为国产操作系统代表，需要自主可控的智能运维能力。Agent 记忆模块能让 OS 助手记住用户偏好和常见问题解决方案，实现个性化、高效的端侧智能服务，减少对云端大模型的依赖，保障数据安全。
