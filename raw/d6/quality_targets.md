# D6 质量目标

本阶段不要求只有一个标准答案，重点是让系统能解释自己的清洗、检索和回答质量。

## 预期清理效果

- 去重后，重复事件、重复工具轨迹、重复偏好不会重复计分。
- 脱敏后，邮箱、手机号、测试账号等敏感内容不会原文出现在长期记忆和最终回答中。
- 冲突处理后，能说明采用最新记录、用户明确表达、默认配置、临时指令、人工复查中的哪一种策略。
- 检索后，回答所用证据能追溯到清洗后的记忆或知识。
- 评测后，至少能输出命中、误命中、泄露风险、冲突未处理、回答偏好不一致五类问题统计。

## 建议指标

| 指标 | 目标 |
|------|------|
| 有效记录保留率 | 不低于 85% |
| 重复记录清除率 | 不低于 95% |
| 敏感信息脱敏率 | 100% |
| 明确偏好命中率 | 不低于 80% |
| 冲突项可解释率 | 不低于 80% |
| 工具失败误判为知识失效 | 0 次 |
